<h2 class="tle">Typography<h2>
<p>Refers to the styling and arrangement of text on a
web page using Cascading Style Sheets. It
encompasses all the properties and techniques used
to control how text appears to the user, ensuring
legibility, readability, and visual appeal.
</p>
</div>


<div class="columntle4">
<h2 class="tle">Key aspects of typography in CSS <h2>

<p>font-family: Specifies the typeface:<br>
When using multiple values, the fontfamily list of font families defines the priority
in which the browser should choose the font
family.<br>
The browser will look for each family on the
user's computer and in any @font-face
resource.<br>
The list is prioritized from left to right: it will use
 the first value if it's available, or go to the next
 one, until the end of the list is reached.
 The default font family is defined by the
 browser preferences.</p>

<p>
normal - Sequences of spaces are combined into one.

pre-line - Only new lines are preserved. Sequences

new lines but also when reaching the end of

<div class="columntle3">

<h2 class="tle">CSS Selectors<h2>
<p>CSS selectors are used to “find” or “select” the HTML
elements that you want to style.<br>CSS selectors are divided into 6 types:</p>
<ol>
<li>Class selector</li>
<li>Element selector or Tag selector</li>
<li>ID selector</li>
<li>Universal selector</li>
<li>Group selector</li>
<li>Attribute selector</li>
</ol>

<h2 class="tle">Element Selector<h2>

<p>selects HTML elements (p, div, h1, etc.) and applies
 CSS to them</p>
<image src="thirdlesson5pic1.JPG">
</div>


<div class="columntle4">

<h2 class="tle">ID Selector<h2>
<ul>
<li>is a style applied to one element in a page.</li>
<li>normally used once, usually in a page title or the
 navigation part of the page.</li>
<li>defined by “#” and uses id attribute of the HTML
 element.</li>
<li>ID selector selects the HTML element with a unique
identifier (id) and adds CSS to it.</li>
<li>it is not advisable to start an id selector with a
 number because some browsers do not support it.</li>
<li>the ID selector is unique and selects one unique
element</li>
</ul>
<image src="thirdlesson5pic2.JPG">
</div>


<div class="columntle5">
<h2 class="tle">Universal Selector<h2>
<p>The universal selector selects every single HTML
 element on the page. It is written using the asterisk
 ( * ) character </p>
<image src="thirdlesson5pic3.JPG">
</div>
</div>




<div class="columntle3">
<h2 class="tle">Group Selector<h2>

<p>The group selector allows you to select multiple elements and apply the same style to all of them</p>
<image src="thirdlesson5pic4.JPG">
</div>
<div class="columntle4">

<h2 class="tle">Attribute Selector<h2>
<p>The attribute selector selects elements based on specific
 attribute values. The syntax for the attribute selector is:
 Element[attribute]</p>
<image src="thirdlesson5pic5.JPG">
</div>


<div class="columntle5">
<p>
When to use class selector:<br>
Use a class selector when you want to apply the same style
to multiple elements or when elements share a common
category or purpose. Use class when styling many
elements that share the same style — even if they have
different tags.<br>
When to use ID selector:<br>
Use an ID selector when styling a unique element that
appears only once on a page (like a header, footer, or main
section).<br>
When to use element/tag selector:<br>
Use an element selector to style all occurrences of a specific
HTML tag (like all p or h1).<br>
When to use universal selector:<br>

Use the universal selector when you want to style all
elements on the page — for example, to reset margins,
padding, or box-sizing.<br>
When to use group selector:<br>
Use a group selector when different elements share the same
styling — to avoid repeating code.<br>
When to use attribute selector:<br>

Use an attribute selector when you want to style elements based
on their HTML attributes or attribute<br>
</p>
</div>
</div>

<image src="thirdlesson6pic1.JPG">

<div class="columntle4">

<h2 class="tle">Pseudo-Classes and Links<h2>
<ul>
<li>Recall that classes are user-defined selectors that
can be used to control individual HTML element
formatting</li>
<li>Pseudo-classes are defined by a colon and are not
user-defined. Specific pseudo-classes can be used
on specific HTML elements only</li>
<li>The anchor element has four pseudo-classes: link,
visited, hover and active. Each one affects the
behavior of the anchor element.</li>
<li>The link pseudo-class is for hyperlinks that have
NOT yet visited.</li>
<li>The visited pseudo-class is for visited hyperlinks.</li>
<li>The hover pseudo-class is for hyperlinks when the
mouse is hovered over them.</li>
<li>The active pseudo-class is for hyperlinks when
they are clicked.</li>
</ul>
</div>


<div class="columntle5">
<ul>
<li>The order in which the pseudo-classes are stated is
important.
</li>

<li>Take note of the LVHA or Link, Visited, Hover,
Active is the order of the pseudo-classes.</li>
</ul>
<p>The value that you place inside {…} will be
displayed whenever a pseudo-class property is
performed. You can set the color of the text or
element, its background, its decoration and many
other characters.</p>
