<h2 class="tle">What is CSS?<h2>
<ul>
<li>CSS stands for Cascading Style Sheets</li>
<li>CSS describes how HTML elements are to be displayed on screen, paper, or in other media.</li>
<li>CSS saves a lot of work. It can control the layout of multiple web pages all at once</li>
<li>External stylesheets are stored in CSS files.</li>
<li>CSS is the language we use to style a Web page.</li>
<li>CSS separates content from design, making websites easier to maintain and more attractive.</li>
</ul>

<div class="columntle4">
<h2 class="tle">Structure of CSS<h2>
<h2 class="tle">Selector</h2>
<p>the HTML element you want to style; is a pattern used to "find" or "select" the HTML elements on a webpage to which a set of CSS rules will be applied</p>

<h2 class="tle">Property</h2>
<p>the style attribute you want to modify; is a named characteristic or attribute that defines a specific aspect of how an HTML element should be displayed or rendered by the browser. Properties are used within CSS declarations to apply styles to selected elements</p>
</div>

<div class="columntle5">

<h2 class="tle">Value</h2>
<p>the weight of the attribute you want to apply to the element property; is the specific setting or configuration assigned to a CSS property. A property and its value form a CSS declaration, which dictates how an element should be styled</p>

<h2 class="tle">Declaration</h2>
<p>is a single instruction that defines a specific style for an element</p>

<h2 class="tle">Declaration Block</h2>
<p>fundamental part of a CSS rule. It is a block of code that contains one or more CSS declarations, which define the styling rules for the selected HTML element(s).</p>

</div>
</div>



<div class="row">
<div class="columntle3">

<h2 class="tle">What is a <span class="green">Stylesheet</span>?<h2>

<p>A set of instructions to a Web browser on how to display various elements on a Web page is known as style sheet.<br>
A style sheet suggests the browser to display a page in a particular way, how the pages should be displayed in the browser.

<br><br>
Every CSS (whether it is contained in a .css file, or embedded in the head element of an HTML document) is a series of instructions called statements</p>
</div>

<div class="columntle4">

<h2 class="tle">What does a <span class="green">Statement</span> do?<h2>

<ul>
<li>It identifies the elements in an HTML document that it affects.</li>
<li>It tells the browser how to draw these elements.</li>
</ul>

<p>Elements are considered to be paragraphs, links, lists items, and so on. In technical terms HTML terms, an element is anything marked up inside HTML tags.</p>

</div>

<div class="columntle5">
<p>Style sheets are just text files, or text embedded in the head of an HTML document, that help separate content from appearance. The content of the page goes into an HTML file. And the appearance goes into a style sheet.</p>

<h2 class="tle">Property</h2>
<p>the style attribute you want to modify; is a named characteristic or attribute that defines a specific aspect of how an HTML element should be displayed or rendered by the browser. Properties are used within CSS declarations to apply styles to selected elements</p>

</div>
</div>

<h2 class="tle">Display Property<h2>
<p>CSS display property allows you to control how an element is displayed</p>

<image src="lesson3tle1.jpg">
</div>


<div class="columntle4">
<h2 class="tle">Block Level Element<h2>

<p>Block level elements take up as much space as possible by default. Each block level element will start a new line on the page, stacking down the page. In addition to stacking vertically, block level elements will also take up as much horizontal space as possible.
</p>

<h2 class="tle">Inline Level Element<h2>

<p>Inline elements display in a line. They do not force the text after them to a new line. An anchor (or link) is an example of an inline element. You can put several links in a row, and they will display in a line.
</p>
</div>

<div class="columntle5">

<h2 class="tle">Why is this important?</h2>
<p>How an element behaves when styled with CSS will change based on the display mode of an element (block vs. inline). Some CSS properties react differently for each display type. We'll learn more about this behavior when we start to lay out pages in CSS<br>
remember that Inline elements cannot contain Block level elements</p>
<image src="lesson3tle2.jpg">

</div>
</div>

<h2 class="tle">CSS Selectors<h2>
<p>CSS selectors are used to “find” or “select” the HTML elements that you want to style. </p>

<p>Css selectors are divided into 6 types:</p>
<ol>
<li>Class selector</li>
<li>Element selector or Tag selector</li>
<li>ID selector</li>
<li>Universal selector</li>
<li>Group selector</li>
<li>Attribute selector</li>
</ol>
</div>


<div class="columntle4">
<h2 class="tle">Element Selector</h2>
<p>selects HTML elements (p, div, h1, etc.) and applies CSS to them</p>
<image src="lesson5tle1.jpg">
</div>



<div class="columntle5">
<h2 class="tle">ID Selector</h2>
<p>is a style applied to one element in a page. It is normally used once, usually in a page title or the navigation part of the page and defined by the hashtag and uses id attribute of the HTML element.
<br>ID selector selects the HTML element with a unique
identifier (id) and adds CSS to it. Though, it is not advisable to start an id selector with a number because some browsers do not support it. The ID selector is unique and selects one unique element</p>
<image src="lesson5tle2.jpg">

</div>
</div>

<h2 class="tle">Pseudo-Classes and Links</h2>
<p>Recall that classes are user-defined selectors that can be used to control individual HTML element formatting. Pseudo-classes are defined by a colon and are not user-defined. Specific pseudo-classes can be used on specific HTML elements only. The anchor element has four pseudo-classes: link, visited, hover and active. Each one affects the behavior of the anchor element.</p>
<ul>
<li>The link pseudo-class is for hyperlinks that have NOT yet visited.</li>
<li>The visited pseudo-class is for visited hyperlinks.</li>
<li>The hover pseudo-class is for hyperlinks when the mouse is hovered over them.</li>
<li>The active pseudo-class is for hyperlinks when they are clicked.</li>
